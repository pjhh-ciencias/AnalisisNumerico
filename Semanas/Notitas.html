Aritmetica Computacional

Problemas bien planteados

Definición: Un problema matemático se dice bien planteado (en el sentido de Hadamard) si existe una solución del problema, es única y depende continuamente de los datos del problema.

Un problema matemático se dice mal planteado si no esta bien planteado.

Un problema bien planteado es aquel que se pueden resolver razonablemente bien con la ayuda de una computadora. Sobre todo cuando la solución no se puede hallar de forma analítica. Para ello se necesita:

  1. Desarrollar metodos (algoritmos) para resolver ecuaciones numericamente.
  2. Implementar los métodos en un software númerico
  3. Ejecutar el software (o programa) en una computadora para simular numericamente un proceso
  4. Representar los resultados obtenidos en alguna forma comprensible tales como visualizaciones gráficas.
  5. Interpretar y visualizar los resultados obtenidos (compararlos con la realidad), repetir parte del proceso númerico o todo el proceso si es necesario.

Discretización de un problema matemático

Para que un problema matemático se pueda resolver con la ayuda de una computadora a traves de métodos númericos es necesario transformarlo a su vez en un problema computacional. El proceso de transformación es llamado discretización del problema matemático.

Para discretizar un problema matemático se siguen algunas de las siguientes estrategias:

  1. Reemplazar espacios de dimension infinita con espacios de dimensión finita.
  2. Reemplazar procesos infinitos con procesos finitos (por ejemplo una integral o una serie infinita con una suma finita)
  3. Reemplazar ecuaciones diferenciales (ordinarias o parciales) con ecuaciones algebráicas.
  4. Reemplazar problemas no lineales en problemas problemas lineales.
  5. Reemplazar problemas de orden alto en sistemas de orden bajo.
  6. Reemplazar funciones complicadas en funciones simples (como polinomios)
  7. Reemplazar matrices generales con matrices que tiene formas simples (como triángulares)

Fuentes de aproximación

Se distinguen al menos 5 fuentes de aproximación que pueden generar errores que afectan los resultados de problema científico.

  1. De aproximación: Durante el proceso de modelación es posible que se omitan o simplifiquen algunas consideraciones importantes del problema de estudio. (e.g. fricción, viscosidad, resistencia del aire, entre otros).
  2. De mediciones empiricas: Ocurre cuando hay necesidad de obtener datos en el laboratorio o en el campo experimental , pues todos los instrumentos de medición hasta ahora inventados son de precisión infinita (errorres sistematicos). La exactitud de los datos puede estar fuertemente limitado por el tamaño de la muestra o las lecturas pueden estar sujetas a ruidos aleatorios (errores accidentales). Por ejemplo cuando hay necesidad de medir la temperatura durante una determinada hora del dia, durante un mes o cuando hay necesidad de obtener el censo de población de un país.

Error de redondeo

El error que se produce cuando se utiliza una calculadora o computadora para realizar cálculos con números reales recibe el nombre de error de redondeo. Se presenta porque la aritmetica realizada en una maquina incluye numeros con un solo nuero finito de digitos y esto da como resultado cálculos realizados únicamente con representaciones aproximadas de los números reales. En una computadora, sólo un subconjunto relativamente pequeño del sistema de números reales se usa para la representación de todos los números reales. Este subconjunto sólo contiene números racionales, tanto positivos como negativos, y almacena la parte fraccionaria, junto con una parte exponencial.
