Métodos iterativos clásicos

Existen muchas maneras de interpretar la solución de un sistema de ecuaciones lineales: $$A\vec{x} = \vec{b}$$ como el punto fijo de una transformación $\vec{x} = H\vec{x} + \vec{c}$.

Consideremos una matriz $M \in \mathbb{M}_{n\times n}(\mathbb{R})$ invertible. Con ayuda de esta matriz construyamos la siguiente ecuación de punto fijo: $$\vec{x}= \left(\mathrm{I}_{n \times n} - M^{-1}A\right)\vec{x} +M^{-1}\vec{b} $$ en este caso $H:= \mathrm{I}_{n \times n} - M^{-1}A $ y $\vec{c} := M^{-1}\vec{b} $

Entonces de acuerdo con el teorema anterior, la dificultad radica en encontrar una matriz $M$ tal que: $$\rho(\mathrm{I}_{n \times n} - M^{-1}A) < 1$$

Un grupo de métodos iterativos clásicos se obtienen si descomponemos la matriz de coeficientes $A$ como: $$A= \mathrm{D}- \mathrm{L} - \mathrm{U}$$ donde

  1. $\mathrm{D}$ es una matriz diagonal tal que $$d_{ii}:= a_{ii}, \,\,\,\,\, \forall i=1,\ldots ,n$$

  2. $\mathrm{L}$ es una matriz triangular inferior tal que $$\ell_{ij}:= - a_{i,j}, \,\,\,\,\, \forall i,j=1,\ldots ,n, i>j$$

  3. $\mathrm{U}$ es una matriz triangular superior tal que $$u_{ij}:= - a_{i,j}, \,\,\,\,\, \forall i,j=1,\ldots ,n, i < j$$

Método de Jacobi

El método de Jacobi se obtiene cuando $M=\mathrm{D}$. En este caso, la ecuación de punto fijo es: $$\vec{x}^{(k+1)} = \mathrm{D}^{-1}(L + U)\vec{x}^{(k)} + \mathrm{D}^{-1}\vec{b}$$

Método de Gauss-Seidel

El método de Gauss-Seidel se obtiene cuando $M=\mathrm{D}-\mathrm{L}$. En este caso, la ecuación de punto fijo es: $$\vec{x}^{(k+1)} = \left(\mathrm{D} - L \right)^{-1} U \vec{x}^{(k)} + \left(\mathrm{D} - L \right)^{-1}\vec{b}$$
