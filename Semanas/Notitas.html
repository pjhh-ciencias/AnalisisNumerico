Nota

Si una matriz $A$ se factoriza como: $A=\mathrm{LU}$, entonces al sustituir en $A\vec{x}= \vec{b}$, se tiene que $$\left(\mathrm{LU} \right)\vec{x}= \vec{b}$$ o bien, $$\mathrm{L}\left(\mathrm{U} \vec{x}\right)= \vec{b}$$ haciendo $\vec{y}:= \mathrm{U}\vec{x}$, se tiene que resolver el sistema: $$\mathrm{L}\vec{y}= \vec{b}.$$ Luego, una vez que se tiene el valor del vector $\vec{y}$ se resuelve el sistema: $$\mathrm{U}\vec{x}= \vec{y}$$

Observación

Notemos que los sistemas $\mathrm{L}\vec{y}= \vec{b}$ y $\mathrm{U}\vec{x}= \vec{y}$ son muy faciles de resolver.

Ahora el problema de resolver el sistema de ecuaciones $A=\mathrm{LU}$ se traduce a factorizar una matriz de la forma: $A=\mathrm{LU}$
