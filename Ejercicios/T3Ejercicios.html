Ejercicio

Crea un programa utilizando Python o Scilab que calcule la suma de dos matrices previamente almacenadas en arreglos. El programa debe validar que las dimensiones de las matrices sean las apropiadas para poder llevar a cabo la suma, en caso contrario imprimir un mensaje de advertencia adecuado.

Ejercicio

Crea un programa utilizando Python o Scilab que calcule el producto de dos matrices previamente almacenadas en arreglos. El programa debe validar que las dimensiones de las matrices sean las apropiadas para poder llevar a cabo la suma, en caso contrario imprimir un mensaje de advertencia adecuado.

Ejercicio

Crea un programa utilizando Python o Scilab que determine la transpuesta de una matriz previamente almacenada en un arreglo.

Ejercicio

Sea $A\in\mathbb{M}(\mathbb{R})$, se define y denota la matriz adjunta de $A$ como la matriz $\mathrm{Adj}(A)$ cuyo elemento de la fila $i$ y la columna $j$ es: $$(-1)^{i+j} |A^{i,j}|$$ donde la matriz $A^{i,j}$ es la matriz que se obtiene al eliminar la fila $i$ y la columna $j$ de la matriz $A$ .

Crea un programa utilizando Python o Scilab que calcule la matriz adjunta de una matriz cuadrada previamente almacenada en un arreglo.

Ejercicio

Crea un programa utilizando Python o Scilab que calcule el determinante de una matriz cuadrada previamente almacenada en un arreglo. El programa debe validar que en efecto se tiene una matriz cuadrada.

Ejercicio

Crea un programa utilizando Python o Scilab que calcule el determinante de una matriz cuadrada previamente almacenada en un arreglo. El programa debe validar que en efecto se tiene una matriz cuadrada.

Hint:Utiliza el desarrollo por cofactores.

Ejercicio

Crea un programa utilizando Python o Scilab que dada una matriz cuadrada $A$ previamente almacenada en un arreglo, verifique si la matriz $A$ es invertible utilizando su determinante. Si $\mathrm{det}(A) \not=0$ calcular la matriz inversa de $A$ y en caso contrario mandar un mensaje advertiendo que la matriz no es invertibley detener el programa. El programa debe validar que en efecto se tiene una matriz cuadrada.

Hint:Utiliza el desarrollo por cofactores.
